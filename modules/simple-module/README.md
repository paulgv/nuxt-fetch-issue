## Setup
- Add `simple-module` dependency using yarn or npm to your project
- Add `simple-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'simple-module',

    // With options
    ['simple-module', { /* module options */ }],
 ]
}
```
